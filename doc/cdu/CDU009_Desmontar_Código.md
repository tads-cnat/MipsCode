## 1. Resumo
Ator é capaz de desmontar o código para editar o código novamente.

<br>

## 2. Atores: 
Usuário e visitante.

<br>

## 3. Pré-condições:
Código deve estar montado.

<br>

## 4. Pós-condições: 
Não há.

<br>

| Ator | Sistema |
| :-----------------: | :-----------------: |
