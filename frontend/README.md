# Iniciando com React (Vite)

## Procedimentos
- Va para o diretório do projeto em sua máquina.
- Run ``` cd frontend ```
- Run ``` npm install ```
- Run ``` npm install @mui/material @emotion/react @emotion/styled ```
- Run ``` npm install @fontsource/roboto ```
- Run ``` npm install @mui/icons-material ```
- Run ``` npm run dev ```
